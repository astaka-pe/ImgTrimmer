# ImgTrimmer

## Installation

```
git clone https://github.com/astaka-pe/ImgTrimmer.git
cd ImgTrimmer

# Linux
python3 -m venv .venv
source .venv/bin/activate # source .venv/bin/activate.fish

# Windows
py -3.10 -m venv .venv
.venv/Scripts/activate

pip install -r requirements.txt
```
